🧭 GAME FLOW STRUCTURE
🎮 1. Start Game Screen

Design a clean and engaging start screen with:

Coin / credit indicator

Primary CTA button: “Start Game”

Mode selection:

Normal Battle

Boss / Event Mode

UI Direction:

Arcade feel

Neon glow accents

Animated button feedback

🧩 2. Pokémon Selection / Scan (Tag System)

Create a selection interface that simulates scanning physical tags:

3 slot Pokémon system (Slot 1–3)

Option:

Scan Pokémon tag

Use default Pokémon

Each Pokémon Card should display:

Name

Type (with icon)

Power stat

Skill preview

UX Notes:

Drag & drop or tap to assign Pokémon

Highlight active slots

Visual difference between scanned vs default Pokémon

⚔️ 3. Enemy Encounter Screen

Design a cinematic battle intro:

Player Pokémon vs Enemy Pokémon

Example layout: left (player) vs right (enemy)

Show:

Enemy rarity indicator

Power comparison

Add:

Dramatic transition animation

“Wild Pokémon Appeared!” text

⚡ 4. First Attack Chance (Quick Reaction UI)

Design a quick-time event screen:

Big visual prompt: “First Attack Chance!”

Timing bar / tap zone mechanic

Interaction:

User must tap at the right moment

Feedback:

Success → “First Attack!”

Fail → “Enemy Moves First!”

👊 5. Battle Phase Screen

Design the main combat interface:

Action buttons:

Attack

Special Skill

Battle HUD:

HP bars

Pokémon avatars

Type indicators

Interaction:

Tap / Smash / Hold mechanics

Animated feedback on hit

🎯 6. Core Quick-Time Mechanic

Design a dynamic QTE system:

Circular timing indicator

Expanding/shrinking ring

Prompt text: “SMASH!”

Rules Visualization:

Perfect timing → critical damage

Miss → low damage

UI Style:

High energy

Particle effects on success

🏆 7. Battle Result Screen

Create result summary:

Win / Lose state

Stats:

Damage dealt

Accuracy

Bonus score

CTA:

Continue → Capture Phase

Retry (optional)

🎲 8. Capture Phase (Gacha System)

Design a reward animation screen:

Pokéball-style capture animation

Randomized outcome feel

UI Elements:

Capture success rate indicator

Rare Pokémon highlight effects

🪙 9. Reward / Tag Collection Screen

Design a reward reveal screen:

Show acquired Pokémon

Simulate physical tag output

Elements:

3D card/tag style

“New Pokémon Acquired!” text

CTA:

Save to Collection

Play Again
